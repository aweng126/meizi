## 爬取妹子图片
1. python版本 3.6.0
2. scrapy 通过anaconda安装
3. 下载源码安装完环境之后,执行
```
scrapy crawl mzitu
```
即可
3. 爬取的网站是[妹子图](mzitu.com)
